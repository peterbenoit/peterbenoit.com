This interactive map showcases U.S. states and territories, providing key details such as population, year founded, capital, and area. The map is powered by Leaflet.js, styled with Tailwind CSS, and uses CSV data files. The geographical data is stored in GeoJSON format and processed using Mapshaper. Additional data is provided through a CSV file, which is dynamically loaded and displayed in the popups.
