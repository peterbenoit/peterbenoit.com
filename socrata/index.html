About This Project

This project provides visualizations of public health data sourced from data.cdc.gov. We use various tools like vanilla JavaScript, Bulma for styling, and DataTables for handling large datasets efficiently.

The aim is to make public health data more accessible and understandable through interactive maps, charts, and data tables.
