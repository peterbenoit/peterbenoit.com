Demo Embed

  • Source: Prefill Embeds
  • Also considering: StackBlitz | CodeMirror

The Prefill Embeds offered by CodePen is a way to demo code in a sandbox which does not require hosting the code on their service. Google uses it as well, see here.

				<div
	class="codepen"
	data-prefill='{
		"title": "EXAMPLE DEMO",
		"description": "EXAMPLE OF DEMO",
		"html_classes": ["no-js"],
		"head": "&lt;meta name=&#x27;viewport&#x27; content=&#x27;width=device-width, initial-scale=1&#x27;&gt;",
		"stylesheets": [],
		"scripts": []
	}'
	style="height: 300px; overflow: auto;"
	data-height="300"
	data-default-tab="html,result"
	data-editable="true"
	data-theme-id="light"
>
	<pre data-lang="html">
		[ESCAPED HTML]
	</pre>
	<pre data-lang="scss">
		[CSS,SCSS,SASS,LESS...]
	</pre>
	<pre data-lang="js">
		[JAVASCRIPT,BABEL,TYPESCRIPT,COFFEESCRIPT...]
	</pre>
</div>
