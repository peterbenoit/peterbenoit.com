Standard Buttons - Filled or Outline

How to use

  • Standard buttons can be either filled or outlined and can use any of the 3 button colors - primary green, secondary blue, or tertiary purple - in order of importance
  • Start button labels with a verb
  • Avoid using vague terms such as "Okay" "Yes" "No"
  • Filled buttons should command the most attention
  • Overall screen layout should make it clear which buttons are the most/least important
  • If linking between other pages or linking to something externally, consider using a text button instead of a standard button
  • Don't oversaturate a page with too many buttons
  • Button labels should be all caps
  • Try keeping button label as short as possible
  • Consider using icons to enforce specific actions - icons on the left for main actions and only used on the right for minor actions
  • Outline buttons should be used with standard buttons when presenting users with two actions
