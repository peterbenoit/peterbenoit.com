Dropdowns

How to use

  • There are 3 sizes of dropdowns - regular/medium/small - regular is the default size in all instances, medium/small should be used on smaller screens where the regular size is too big
  • Used for choosing between a list of options
  • Don't use if a user needs to select multiple options (Consider selection modals instead)
  • Don't use as navigation
  • Don't use if the list gets unmanageably long and becomes frustrating to scroll through (Consider selection modals instead)
  • Don't use for information that's already very familiar to users (Ex: entering personal information on a form), consider form fields instead
  • Keep dropdown label in view once a user clicks into the menu below
  • On hover, the menu item background will turn light blue
  • Once a menu item is clicked, dropdown will automatically collapse and display selected option
